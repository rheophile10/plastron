web-editor

A live-editable plastron app — edit the spec on the left, the right side hydrates it as a child segment and shows the result. Same factory (buildWebEditor) the plastron-OS web-editor app uses, rendered standalone.

Wiring: buildWebEditor({ mount: "#webedit" }) hydrates with the COUNTER_EXAMPLE spec; the inner editor parses your JSON spec into cels each tick (so syntax errors degrade gracefully). Try the WEATHER_EXAMPLE by editing main.ts and swapping it in.
