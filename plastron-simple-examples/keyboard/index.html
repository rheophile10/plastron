keyboard

A musical keyboard driven by plastron's sound segment. Press keys on your keyboard (or click) — each key triggers sound.play-tone. Same cel doom uses for SFX dispatch.

click anywhere on the page to enable sound (browser autoplay policy)

Wiring: this page hits resolveFn(state, "sound.play-tone") and calls it with {freq, duration, type, gain}. The same sound segment ships in the kernel; the doom example uses its sound.play-pcm sibling for SFX.
